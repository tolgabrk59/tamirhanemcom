### 1.1 Araç Servis Bulma Platformları

### 2.1 RepairPal (ABD - En Gelişmiş Model)
- **URL:** https://www.repairpal.io
- **Kuruluş:** 2007 (David Sturtz & David Esser)
- **Satın Alan:** Yelp (2024)
- **Finansman:** $38.2M toplam

**Sunulan Özellikler:**

1. **Fair Price Estimator**
   - 500+ araç markası için uyumlu
   - Zip code bazlı fiyat hesaplama
   - Aylık 500.000+ ziyaretçi
   - MOTOR veritabanı entegrasyonu

2. **RepairPal Certified Network**
   - 3.800+ bağımsız servis
   - Katı kalifikasyon süreci:
     * ASE sertifikası zorunlu
     * Müşteri memnuniyeti 3x sektör ortalaması
     * Net Promoter Score ölçümü
     * 12 ay/12.000 mil garanti
     * Şeffaf fiyatlandırma

3. **Shop Referral System**
   - CarMax, USAA, Consumer Reports ortaklıkları
   - Ortalama müşteri değeri: $800-$1.000

- Servis müşteri başına çok küçük kesinti (B2B)
- Tüketici: Mekanik danışması ($9/soru)

**Farklılaştırıcı Unsurlar:**

- Yaygın partner ağı

## BÖLÜM 3: TÜRKIYE PAZARI ANALİZİ

### 3.1 Pazarın Büyüklüğü

**2024-2025 Veri:**
- İkinci el araç devresi: 10.054.915 adet (Ocak-Kasım 2024, %3.6 artış)
- Sıfır araç pazarı projeksiyonu: ~1.5M adet (rekor)
- 2025 toplam motosiklet devri: 10.5M adet

**Potansiyel Servis Pazarı:**
- Her araç ortalama 2-3x/yıl servis geçirir
- 10.5M x 2.5 = **26.25 milyon servis işlemi/yıl**

### 3.2 Araç Muayene Takvimi (Periyodik Bakım)

**Hususi (Özel) Otomobiller:**
- İlk 3 yıl: Muayeneden muaf
- Sonrasında: Her 2 yılda bir

**Ticari Araçlar:**
- Yılda 1 kez (zorunlu)

**Hazır İşletme Fırsatı:**
- Belirlenebilir muayene takvimi
- Otomatik hatırlatma = Yüksek dönüşüm
- TÜVTÜRK randevu entegrasyonu

### 3.3 Mevcut Rekabet Durumu

**Gözlem 1:** Fragmentasyon
- Armut: Genel hizmet (oto servis bir kategori)
- Rabam: Sadece İstanbul, sınırlı hizmet sayısı
- Arabam.com: Esas ilan pazarı (Garaj ek feature)
- Servis yazılımları: B2B (araç sahibi değil)

**Gözlem 2:** Eksik Noktalar
- Hiçbir platform İndeksleme + Fiyat Tespiti + Servis Bulma + OBD entegrasyonunu birleştirmiyor
- Hiçbir platform "Bakım Takibi" + "Muayene Takvimi" kombinasyonunu sunmuyor
- DIY + Profesyonel servis ayrımı yok
- Yedek parça entegrasyonu eksik
- Muhasebe/finansal takip eksik

**Gözlem 3:** Coğrafi Sinırlama**
- Rabam sadece İstanbul
- Armut yaygın ama şehirsel
- Kırsal alanlar öne çıkmıyor

## BÖLÜM 4: TamirHanem İçin REKABET AVANTAJI

### 4.1 Platform Tasarımı Önerileri

#### **Eksik Olan Özellikler (Türkiye'de ilk)**

1. **Entegre Araç Bakım Defteri**
   - Araç sahibi perspektifi: "Benim araç tarihi"
   - Tüm bakım geçmişi tek yerde
   - Otomatik muayene hatırlatmaları (TÜVTÜRK takvimi)
   - Yedek parça değişimi takibi

2. **Akıllı Fiyat Karşılaştırması**
   - "Benim servisim kaçta tutulmalı?" (AutoMD modelinden adaptasyon)
   - Zip code + araç modeline göre adil fiyat aralığı
   - Türkiye MOTOR-like veritabanı oluştur (Oto Sanayicileri Derneği ortaklığı)

3. **OBD Entegrasyonu**
   - DtcFix integrasyon veya kendi hata kodu veritabanı
   - "Bu hata hangi servisler çözebilir?" önerisi
   - DIY destek: YouTube video tavsiyesi

4. **Yedek Parça Bağlantısı**
   - Servis önerisine "parça nerede alırsın" bilgisi
   - Yedekparca.com.tr, Aloparca entegrasyonu
   - OEM vs. Aftermarket karşılaştırması

5. **Araç Değer Takibi**
   - "Benim araç şu anda kaç para?" (Arabam Kaç Para alternatifi)
   - Bakım kalitesi = Değer artışı korelasyonu
   - Satış önerileri: "Bu araç şu anda satılabilir mi?"

---

### 4.2 Kullanıcı Tarafı Özellikler

#### **Araç Sahipleri İçin:**

1. **Bakım Takvimi Yönetimi**
   ```
   - Otomatik hatırlatma (SMS/Push)
   - Muayene tarihleri (TÜVTÜRK integrasyon)
   - Sigorta yenileme tarihleri
   - Trafik tescil yenileme
   ```

2. **Servis Bulucu + Karşılaştırma**
   ```
   - İhtiyacın tanımla: "Motor yağı değişimi"
   - Yakındaki servisler: Harita görünümü
   - Fiyat aralığı: Adil fiyat = Yeşil, Pahalı = Kırmızı
   - Kullanıcı yorumları: Spesifik hizmet bazında
   ```

3. **Araç Sağlığı Skoru**
   ```
   - Geçmiş bakım verilerine göre
   - 1-100 skor
   - İyileştirme önerileri
   ```

4. **Satış Danışmanlığı**
   ```
   - "Bu araç şu anda kaç para eder?"
   - Satış zamanı önerisi (sezonsal)
   - Ön hazırlık önerileri
   ```

---

#### **Servis İşletmeleri İçin:**

1. **Talep Oluşturma**
   ```
   - Armut-like: Müşteriler sistem üzerinden talep yaratır
   - Otomatik tekliflendirilme
   - Premium görünürlük paketi
   ```

2. **Müşteri Yönetimi**
   ```
   - Randevu sistemi
   - SMS/Push gönderim
   - Muhasebe entegrasyonu (Servisbir-like)
   - Bakım geçmişi depolama
   ```

3. **Pazar İstihbaratı**
   ```
   - "Bölgede talep: Motor yağı değişimi"
   - Ortalama fiyat: Türkiye ortalaması vs. bölgesel
   - Rakip analiz
   ```

4. **Müşteri Sadakati Araçları**
   ```
   - Periyodik hatırlatmalar (sistem yönetiyor)
   - Kupon sistemi
   - Referral bonusu
   ```

---

### 4.3 Monetizasyon Modeli Önerileri

#### **Model 1: Çoklu Kaynak (Önerilen)**

```
Kullanıcıdan:
├─ Servis müşteri başına: 1-2% (RepairPal-style, şeffaf)
├─ Premium profil (Servisler): ₺299/ay

Reklam:
├─ Yedek parça firmaları: Banner/Sponsored listing

└─ B2B danışmanlık (Servis öğretimi)

Ortaklıklar:
├─ TÜVTÜRK entegrasyonu (Harita önerileri)
├─ ODD/OSD ortaklığı (Veri erişimi)
├─ Sigorta şirketleri (DRP program)
└─ Belki sonradan Fintech (Kredi/Leasing)
```

```

```

## BÖLÜM 5: ÖNCELİKLİ ÖNERİLER (İMPLEMENTASYON SIRASI)

### **FAZE 1: MVP (1-3 AY) - Temel Kompetitif Avantaj**

#### Öncelik 1: Entegre Bakım Defteri + Muayene Takibi
- **Neden:** Hiçbir platformda yok, yüksek kullanıcı tutma
- **Uygulanabilirlik:** Orta
- **Kod Complexity:** Orta
- **Pazarlama Etkisi:** Çok Yüksek (Unique Selling Point)

**Özellikler:**
- Araç sahibi profili: Tüm araçları ekle (plaka, şasi)
- Bakım günlüğü: Servisler otomatik kaydedilsin (servis tarafından)
- TÜVTÜRK API: Muayene takvimi otomatik hatırlatma
- Push/SMS: Muayene tarihi 30 gün öncesinde

**Teknik Stack:**
- Veritabanı: Araç profilleri
- Calendar API: Muayene tarihleri
- TÜVTÜRK webhooks/API (mevcut mi kontrol et)

---

#### Öncelik 2: Servis Bulma + Harita
- **Neden:** Armut, Rabam'dan farklı (tam harita, coğrafi kapsam geniş)
- **Uygulanabilirlik:** Yüksek
- **Kod Complexity:** Düşük
- **Pazarlama Etkisi:** Yüksek

**Özellikler:**
- Hizmet türü seç (Motor yağı değişimi, vb.)
- Lokasyon: Harita üzerinde al
- Filtreler: Açık saatler, yorum puanı, fiyat aralığı
- Detay: Harita, iletişim, yorum, randevu

**Teknik:**
- Google Maps API / Openstreetmap
- Servis lokasyon veritabanı (Armut, Rabam'dan scrape + input)
- Real-time availability (randevu sistemi)

---

#### Öncelik 3: Yorum Sistemi (Hizmet Bazında)
- **Neden:** Kalite çiftliği sağlar, ROI sürücü
- **Uygulanabilirlik:** Orta
- **Kod Complexity:** Düşük
- **Pazarlama Etkisi:** Yüksek

**Özellikler:**
- Hizmet bazında yorum: "Motor yağı değişimi sonrası memnun"
- 1-5 yıldız: Genel puan
- Spesifik puanlar: Fiyat, hız, profesyonellik
- Fotoğraf yükleme

**Teknik:**
- Rating sistem
- Moderation queue
- Fake review detection (iplik kazınması)
